Для запуска игры необходимо выполнить команду:
  npm run serve

и перейти в браузер по ссылке:
  http://localhost:8080/

Игра запустится в режиме для разработчика, для production сборки необходимо выполнить команду:
  npm run dist

после этого в папке с игрой появится еще одна папка dist, содержащая все необходимое.
